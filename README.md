# lubanr-commenter
vim comment plugin

## 安装
----
推荐使用pathogen.vim 安装.

        cd ~/.vim/bundle/
        git clone https://github.com/lubaogui/lubanr-commenter.git

## 使用
----
安装完成后，在.vimrc中设置相关变量

        let g:CompanyName = "公司网站或者名称"
        let g:AuthorName = "姓名"
        let g:AuthorEmail = "youremail@lubanr.com"
